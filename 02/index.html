Sandy Bridge

Ця мікроархітектура процесорів була випущена у 2011 році і є другим поколінням процесорів Intel Core та v1 лінійки Xeon E3. Модельний ряд Sandy Bridge включає різні варіанти процесорів, у тому числі:

  1. Intel Core i7 Sandy Bridge

    • Intel Core i7-2600
    • Intel Core i7-2600S
    • Intel Core i7-2600T
  2. Intel Core i5 Sandy Bridge
    • Intel Core i5-2500K
    • Intel Core i5-2500
    • Intel Core i5-2500S
    • Intel Core i5-2500T
  3. Intel Core i3 Sandy Bridge
    • Intel Core i3-2100
    • Intel Core i3-2100T
    • Intel Core i3-2120
    • Intel Core i3-2120T

Sandy Bridge vs Ivy Bridge

Техпроцес

Процесори Ivy Bridge (v2) використовують 22-нм, тоді як Sandy Bridge (v1) використовує 32-нм техпроцес. Використання тоншого техпроцесу дозволяє Ivy Bridge працювати більш ефективно і споживати менше енергії, ніж Sandy Bridge.

Підтримка PCI-Express

Процесори Sandy Bridge підтримують PCI-Express 2.0, в той час, як Ivy Bridge PCI-Express 3.0

  • Під час використання процесорів Sandy Bridge підтримується PCIe 2.0. Версія PCIe 2.0 забезпечує пропускну здатність до 5 гігабайт на секунду (GT/s) на одну лінію. Однак загальна пропускна здатність системи може змінюватись в залежності від кількості доступних ліній та конфігурації слотів PCIe.
  • З архітектурою Ivy Bridge підтримується PCIe 3.0. Версія PCIe 3.0 досягає до 8 гігабайт в секунду (GT/s) на одну лінію.

Таким чином, відмінність у пропускній здатності може бути важливим при виборі компонентів системи, таких як графічні карти або інші пристрої, які потребують високої пропускної здатності PCIe шини.

Підтримка оперативної пам'яті.

Процесори Sandy Bridge підтримують оперативну пам'ять із частотою 1333МГц, Ivy Bridge 1600МГц. Оперативна пам'ять є зовнішнім компонентом, який з'єднується із процесором через контролер пам'яті. Відмінність пропускної спроможності оперативної пам'яті між поколіннями пов'язана з відмінностями в контролерах пам'яті, реалізованих у цих архітектурах процесорів.
