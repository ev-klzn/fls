JavaScript DOM: Основи та Можливості

JavaScript Document Object Model (DOM) є ключовим компонентом у веб-розробці, що дозволяє динамічно взаємодіяти з вмістом веб-сторінки. DOM представляє собою ієрархічну структуру об'єктів, які представляють кожен елемент, атрибут чи текстовий вузол веб-сторінки. У цій статті ми розглянемо основні поняття та можливості JavaScript DOM.

Основні Поняття

DOM елементи:

DOM представляє веб-сторінку як ієрархію елементів. Кожен тег HTML стає DOM елементом, який можна змінювати та взаємодіяти за допомогою JavaScript.

Дерево DOM:

DOM структура представляється як дерево, де кожен елемент є вузлом. За допомогою JavaScript, можна отримувати доступ до будь-якого вузла і змінювати його.

Робота з Елементами

Отримання Елементів:

Використовуючи методи, такі як getElementById, getElementsByClassName, getElementsByTagName або сучасні querySelector і querySelectorAll, можна отримати доступ до елементів.

Зміна Властивостей:

Змінюйте атрибути та стилі елементів, використовуючи властивості DOM.

Робота з Подіями

Додавання Подій:

Додавайте обробники подій, щоб реагувати на взаємодію користувача.

Робота з Подією:

У функції обробника події можна взаємодіяти з елементами та їхніми властивостями.

Динамічне Створення Елементів

Створення Елементів:

Використовуйте метод createElement для динамічного створення нових елементів.

Додавання до Дерева:

Додавайте нові елементи до дерева DOM за допомогою методу appendChild.

JavaScript DOM надає широкий спектр інструментів для взаємодії з веб-сторінками.

  • document.getElementById(id) Отримання елементу за його ідентифікатором.
  • document.getElementsByClassName(className) Отримання колекції елементів за їхніми класами.
  • document.getElementsByTagName(tagName) Отримання колекції елементів за їхнім ім'ям тегу.
  • document.querySelector(selector) Отримання першого елемента, який відповідає CSS-селектору.
  • document.querySelectorAll(selector) Отримання всіх елементів, які відповідають CSS-селектору.
  • element.innerHTML Отримання або встановлення HTML-вмісту елемента.
  • element.style.property Отримання або встановлення стилів елемента.
  • element.addEventListener(event, handler) Додавання обробника події до елемента.
  • element.removeChild(child) Видалення дочірнього елемента з батьківського.
  • element.appendChild(child) Додавання дочірнього елемента до батьківського.
  • document.createElement(tagName) Створення нового елемента з вказаним ім'ям тегу.
  • element.setAttribute(name, value) Встановлення атрибута елемента з вказаним ім'ям та значенням.
  • element.getAttribute(name) Отримання значення атрибута елемента за вказаним ім'ям.
  • element.classList.add(className) Додавання класу до списку класів елемента.
  • element.classList.remove(className) Видалення класу зі списку класів елемента.
  • element.classList.toggle(className) Додавання або видалення класу відповідно до його наявності.

Це лише кілька прикладів засобів, які вам доступні при роботі з JavaScript DOM. Знання цих методів і властивостей дозволить вам ефективно маніпулювати структурою та вмістом веб-сторінок.
